# AirportInformationSystem

## Backend

### How to run

```bash
python back/main.py
```

If database changed use this command to update models
```bash
python back/lib/generators
```

## Frontend

### How to run

1. Go to front folder
```bash
cd front
```

2. Install dependencies
```bash
npm install
```

3. Run
```bash
npm run dev
```
